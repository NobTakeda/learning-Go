### A Tour of Go
### 6-7~: Exercise: Equivalent Binary Trees
+ 二分木に入っている“数列”が同じかどうかを、ゴルーチンとチャネルを用いて判定するという演習
+ サンプルstructはこちら。Go Treeを使用しているとのこと。  
https://pkg.go.dev/golang.org/x/tour/tree#Tree
+ 使用する場合、`import "golang.org/x/tour/tree"`が必要
```go
type Tree struct {
	Left  *Tree
	Value int
	Right *Tree
}
```
演習の解答例
```go
// 二分木tを中順(Left→Value→Right)に走査し、見つけた値をchへ順番に送信する
func Walk(t *tree.Tree, ch chan int) {
	if t == nil {
		return
	}
	Walk(t.Left, ch)
	ch <- t.Value
	Walk(t.Right, ch)
}

// walkerはWalkの送信完了後にchをcloseするラッパー
// chのcloseは生成側の責任にするのがGoの慣習とのこと
func walker(t *tree.Tree, ch chan int) {
	Walk(t, ch)
	close(ch)
}

func main() {
	ch := make(chan int)
	go walker(tree.New(1), ch)

	// 10個受け取って表示(rangeでもOK。New(1)は10個ちょうど)
	for v := range ch {
		fmt.Println(v) // 1, 2, 3, ..., 10 が出力される
	}
}

// t1, t2が同じ値列を保持しているかを判定する。それぞれをチャネルにストリーム化し、逐次比較
func Same(t1, t2 *tree.Tree) bool {
	ch1 := make(chan int)
	ch2 := make(chan int)

	go walker(t1, ch1)
	go walker(t2, ch2)

	for {
		v1, ok1 := <-ch1
		v2, ok2 := <-ch2
        // 片方だけ終了（長さ違い）
		if ok1 != ok2 {
			return false
		}
        // 両方終了＝完全一致
		if !ok1 {
			return true
		}
		if v1 != v2 {
			return false
		}
	}
}
```
